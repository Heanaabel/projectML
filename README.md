#Projet d'analyse et d'application de méthodes de Machine Learning

Ce projet propose une analyse détaillée du dataset Breast Cancer, incluant :

Prétraitement des données : nettoyage, normalisation, imputation des valeurs manquantes…
Application de méthodes de Machine Learning supervisées pour prédire si une tumeur est bénigne ou maligne.
Métriques utilisées : exactitude (accuracy), précision, rappel, F1-score.
Comparaison de différents modèles : Random Forest, SVM, KNN, etc.

Le projet contient également plusieurs autres outils de ML développés au fur et à mesure de mon apprentissage des principales librairies de Machine Learning :

LDA (Linear Discriminant Analysis) sur le dataset Wine
KMeans (clustering non supervisé) sur le dataset Iris
Decision Tree (arbre de décision supervisé) sur le dataset Iris
Régression linéaire sur le dataset California Housing

Chaque fichier contient une fonction run_<nom>() permettant d’exécuter l’analyse correspondante, et main.py permet d’exécuter tous les outils en une seule fois.
