# ML AI Project - Paris Saclay Application

Ce projet contient plusieurs tools de Machine Learning dont j'ai développé au fur et à mesure de mon apprentissage des librairie du ML:

- LDA sur le dataset Wine
- KMeans sur le dataset Iris
- Decision Tree sur le dataset Iris
- Régression linéaire sur le dataset Housing

Chaque fichier contient une fonction `run_<nom>` et `main.py` permet de tout exécuter.
